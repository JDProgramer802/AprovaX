# 🤖 AprovaX Bot - WhatsApp

Bot de WhatsApp multi-grupo para aprobación automática de solicitudes de unión en lotes.

**Desarrollado por Andres 💠**

---

## 📋 Características

- ✅ Aprobación automática de solicitudes en lotes
- ⏱️ Delay configurable entre lotes (3000ms por defecto)
- 📊 Mensajes de progreso en tiempo real
- 🔐 Autenticación mediante QR desde terminal
- 🛡️ Manejo de errores robusto
- 📱 Compatible con múltiples grupos simultáneamente

---
